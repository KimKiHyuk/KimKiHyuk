## 개발 내역

- IOS / Android 빌드 및 배포
- Firebase FCM 푸시메세지 처리 기능 개발
- SMS기반 인증 기능 개발
- UI 및 비즈니스 로직 전체 개발
- Provider를 활용한 상태관리패턴 적용
- Key-value 데이터베이스를 활용한 데이터 제어 로직 개발

## 주요 성과

- 모바일 앱 아키텍처 설계 및 실시간 알람 서비스 구현
- Provider 패턴 도입을 통한 앱 반응성 향상
- 백엔드 서비스와 연동 및 앱에서 사용자 인증 로직 구현
