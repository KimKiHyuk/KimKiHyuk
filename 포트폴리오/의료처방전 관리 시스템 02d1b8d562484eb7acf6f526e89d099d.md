# 의료처방전 관리 시스템

의료 처방전을 모바일 기기를 통해 확인 및 복약지도를 도와주는 서비스

> 사용한 기술
- Android
- JAVA
- OpenCV

🏨 수행 기관 : 소프트웨어 마에스트로

🗓️ 개발 기간 : 3개월

💾 소스코드 :  분실 (계정 삭제로 repository 삭제됨)

---

## 개발 내역

- NDK 를 사용하여 모바일에서 opencv 연동
- Contours 추출을 통한 처방전(사진) 정보 추출 모듈 개발(opencv, ocr)
- 안드로이드 앱 개발 참여
